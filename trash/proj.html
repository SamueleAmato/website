Infrastruttura Tattica Decentralizzata
Project Preview

Il progetto è un'infrastruttura hardware e software progettata per le comunicazioni tattiche off-grid. Sfruttando i microcontrollori ESP32 e il protocollo radio LoRa (Low Power, Long Range), ho creato una rete Mesh decentralizzata e resiliente, priva di un single point of failure.

A differenza dei sistemi tradizionali, questa rete non dipende da server centrali o connessioni internet. Ogni nodo si comporta simultaneamente da client e da router: i dati "saltano" di dispositivo in dispositivo formando una ragnatela auto-riparante. Il tutto è monitorabile tramite una dashboard web custom che gestisce allarmi e messaggistica.

Cybersecurity Fisica: Il protocollo Anti-Tamper

Uno dei limiti dei sensori distribuiti sul campo è la possibilità di compromissione fisica. Per risolvere questo problema, ho implementato un protocollo anti-tamper (Dead Man's Switch). Se un nodo viene aperto o rimosso violentemente, un interrupt hardware azzera istantaneamente le chiavi AES dalla memoria flash e disattiva il dispositivo.

#include <Preferences.h>

Preferences preferences;
const int TAMPER_PIN = 4; // Pin collegato al sensore di apertura case

// Interrupt Service Routine (ISR) ad altissima priorità
void IRAM_ATTR tamperAlert() {
    // 1. Notifica immediatamente la rete Mesh
    sendEmergencyPacket("CRITICAL: NODE_COMPROMISED");

    // 2. Azzeramento crittografico (Zeroing)
    preferences.begin("crypto", false);
    preferences.putString("aes_key", "00000000000000000000000000000000");
    preferences.end();

    // 3. Brick logico del dispositivo
    while(true) {
        // Il microcontrollore è bloccato e non trasmetterà più
    }
}

void setup() {
    pinMode(TAMPER_PIN, INPUT_PULLUP);
    
    // Associa l'evento fisico all'esecuzione immediata dell'ISR
    attachInterrupt(digitalPinToInterrupt(TAMPER_PIN), tamperAlert, FALLING);
}

void loop() {
    // ... Logica di routing e ricezione dei pacchetti LoRa ...
}

L'intera architettura è stata sviluppata con l'obiettivo di renderla 100% Open Source, per permettere a chiunque di studiarla e implementare reti private a bassissimo costo in scenari critici o di emergenza.
